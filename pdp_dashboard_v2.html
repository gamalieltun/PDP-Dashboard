📋 Step-by-Step: Connect Google Sheets API
Follow these steps once — then paste your credentials below to load live data.
  1. Open Google Cloud Console
    Go to console.cloud.google.com. Sign in with your Google account. Click Select a project → New Project. Name it anything (e.g. PDP Dashboard) and click Create.
  2. Enable the Google Sheets API
    In the left menu go to APIs & Services → Library. Search for Google Sheets API and click Enable.
  3. Create an API Key
    Go to APIs & Services → Credentials → Create Credentials → API Key. Copy the key that appears. Optionally click Restrict Key and limit it to Google Sheets API only for security.
  4. Get your Spreadsheet ID
    Open your Google Sheet. The URL looks like:
    https://docs.google.com/spreadsheets/d/SPREADSHEET_ID/edit
    Copy the long ID string between /d/ and /edit.
  5. Make the Sheet viewable
    In your Google Sheet click Share → Change to anyone with the link → Viewer. The API key only works on sheets that are publicly readable.
  6. Paste credentials below and click Load
    Enter your API Key and Spreadsheet ID in the fields below. The dashboard will fetch all sheets live. No backend needed.
